## Introduction

> Ragasは、大規模言語モデル（LLM）アプリケーションの評価を強化するツールを提供するライブラリです。LLMアプリケーションを容易かつ確信を持って評価できるよう設計されています。


## チュートリアルやってみる

```
$ uv init test-ragas-project
```

ragasを追加

```
$ uv add ragas
```
